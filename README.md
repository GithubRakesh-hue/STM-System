# 🎓 Student Test Management System (STM-System)

A lightweight, interactive, menu-driven Python application designed to effortlessly manage student records, track test scores, and generate comprehensive performance reports. Perfect for educators looking for a fast, terminal-based tracking tool! 🚀

---

## ✨ Features

This program features a fully functional loop with 10 powerful options to manage your classroom:

*   **📋 View Roster:** Display a complete list of all students and their Roll Numbers.
*   **➕ Add Test Scores:** Dynamically add new test names and marks for any student.
*   **🔍 Check Individual Records:** Look up a specific student's test history and scores.
*   **📊 Calculate Averages:** Instantly calculate the average marks for a specific student.
*   **🏆 Find Test Toppers:** Identify the highest-scoring student for each individual test.
*   **🌟 Batch Topper:** Discover the absolute top-performing student across all tests.
*   **📉 Needs Improvement:** Identify the student with the lowest average score in the batch.
*   **🎯 High Achievers:** Instantly filter and display students maintaining an average above 80.
*   **📜 Generate Report Cards:** Print beautifully formatted report cards including Total Marks, Average, and Letter Grades (A, B, C).
*   **📈 Batch Performance:** View overall statistics for the entire class, including total headcount, class average, and extreme scores.
